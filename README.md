# Pipe | GPT
Pipe your content to gpt directly from the command line.

## Set the open api key env var in windows powershell
$env:OPENAI_API_KEY = 'sk-12345abc'

##  Set the open api key env var in linux
OPENAI_API_KEY='sk-12345abc'
